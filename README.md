# 🚀 Taskora

**Organize your day, effortlessly.**

Taskora is a modern, responsive To-Do web application designed to help users manage tasks efficiently with a clean UI, authentication system, and persistent storage using LocalStorage.

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 👤 Profile Management
- 📝 Add, Edit, Delete Tasks
- 🔍 Task Filtering & Search
- 📊 Task Status Tracking (Pending / Completed)
- 💾 Data stored using LocalStorage
- 🔔 Toast Notifications for actions
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- LocalStorage (Client-side storage)(no backend storage required)
